Cosmos-UI Release v0.4.0

The latest Cosmos-UI release is now live!

Running the UI with the binary release is super simple. Download the zip file, extract it, then double click the ATOM icon. Don’t forget to save your mnemonic, even if this is just on a test network.

NOTE: DO NOT ENTER YOUR FUNDRAISER SEED. THIS SOFTWARE HAS NOT BEEN AUDITED. NEVER ENTER YOUR FUNDRAISER SEED 12 WORDS ONTO AN ONLINE COMPUTER.

Since the Cosmos-UI is still in pre-alpha, we advise users to exercise vigilance in taking care of not inputing their fundraiser seeds into the application. We have disabled that option in this release, so it is relatively safer than the prior one.

  • Get the Cosmos-UI v0.4.0 binary for macOS and Linux: https://github.com/cosmos/cosmos-ui/releases/tag/0.4.0
  • Get the Cosmos-UI v0.4.1 binary for Windows: https://github.com/cosmos/cosmos-ui/releases/tag/0.4.1 (Fixed crash on startup issue)

What’s New

  • Bonding tokens. The new interface uses sliders instead of a basic text input. Improvements were also made to the user experience in bonding and unbonding from validators.
  • Block Explorer. It’s now possible to search for and browse through blocks.
  • Gaia-2 Testnet. The Cosmos-UI is now built on gaia-2 testnet, which means that the fermion faucet is compatible with it. Users can now make transactions instead of having no coins to test with as was the case in version 0.3.0.

Fun fact: There are currently 74 validators on gaia-1 and 26 on gaia-2.
