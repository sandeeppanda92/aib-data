Cosmos UI Release

Greetings Cosmonauts!

Today we are releasing the first alpha version of Cosmos UI for Mac, Linux, and Windows. Until now, all interactions with the Cosmos Gaia testnet have been via the command line. With this release, we invite you to join Gaia via a gorgeous UI and play around with sending and receiving testnet tokens (fermions). See the release notes for details on how to get some coins.

In this release, you’ll be able to:

  • Create a Cosmos account
  • Restore your account through seed
  • Send/receive tokens on gaia-1 testnet

Lots of great software is in the works. You can expect frequent updates from us as we push through each release on our way to launch. Last week, we released the first public alpha of the Cosmos UI.

Stay tuned for many more features, coming soon!
